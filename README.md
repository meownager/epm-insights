# EPM Insights

EPM Insights is a local-first project performance audit tool for engineering project managers.

The goal is to compare proposal expectations against actual project data, evaluate project health using clear audit metrics, and generate practical reports that support project reviews, post-mortems, and continuous improvement.

## Current Focus

The first phase is the audit engine. Before building a polished app, the project will define how project performance should be measured from a human project management perspective.

The audit engine will focus on:

- Budget performance
- Hours performance
- Resource usage
- Rate and labor mix impact
- Schedule and deadline position
- Billing and balance status
- Change order impact
- Project health scoring
- Final audit findings and recommendations

## Project Principles

- Keep real company data private and local
- Use synthetic data for demos, testing, and public documentation
- Build with free and open-source tools
- Make the analytics logic transparent and explainable
- Use SQL and Python as core learning tools
- Add machine learning only after the audit framework is reliable
